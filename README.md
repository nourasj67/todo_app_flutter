# 📝 Todo App (Task Management)

A professional and clean **Flutter** application for managing daily tasks. This app helps users organize their time by adding, updating, and archiving tasks with ease.

## 🚀 Features (المميزات)
* **Local Storage:** Uses **Sqflite** to save tasks permanently on your device.
* **State Management:** Built with **Cubit/Bloc** for smooth performance and clean code.
* **Task Categories:** Separate tabs for New, Done, and Archived tasks.
* **Bottom Sheet Input:** Easily add new tasks with a date and time picker.
* **Interactive UI:** Swipe to delete or archive tasks with a modern interface.

## 🛠️ Tech Stack (التقنيات المستخدمة)
* **Flutter** (Framework)
* **Dart** (Language)
* **Sqflite** (Local Database)
* **Cubit/Bloc** (State Management)
* **Conditional Builder** (UI handling)

## ⚙️ How to Run (كيفية التشغيل)
1.  Clone this repository:
    ```bash
    git clone [https://github.com/nourasj67/todo_app_flutter.git](https://github.com/nourasj67/todo_app_flutter.git)
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the app:
    ```bash
    flutter run
    ```
